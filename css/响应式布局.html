响应式布局

1.在不同的设备上正常使用 2.一般要处理屏幕大小的问题

主要方法

1.隐藏 + 折行 + 自适应空间 2.rem： 通过在html标签上指定font-size，css样式中单位写rem，如： html{ font-size: 20px; } .container{ width: 10rem; height: 10rem; } viewport: 通过在viewport中的width写成固定值，比如width = 320， 所有设备宽度都会为320， 不同设备上留白会不同 media query: 通过@media(max-width) {} 去修改样式.
