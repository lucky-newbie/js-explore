选择器的解析和性能

css性能优化

1.浏览器从右向左查找css，所以使用合适的选择器，减少查询次数 2.使用css preload， 提前加载css 3.避免使用消耗性能的属性， 比如box-shadow border-radius transform 4.避免重排 5.避免重绘 6.去除无用的css/压缩css内容/部分属性进行合并书写
浏览器解析查找选择器为从右向左方向开始找
