总结

1.实现css动画的方式有几种？

transition animation keyframes

2.过渡动画和关键帧动画的区别

a.过渡动画需要有元素状态的改变 b.关键帧动画不需要元素状态改变 c.关键帧动画控制的更加精细(播放方向、状态、次数等)

3.如何实现逐帧动画

a.使用关键帧动画 animation b.通过使用animation-timing-function： steps(1)去掉补间动画

4.CSS动画性能

1.性能不坏 2.部分情况下优于JS 3.但JS可以做到更好 4.部分高危属性， 如box-shadow， 变化阴影时会耗费性能

5.JS实现动画效果

JS动态修改css样式,setTimeout/setInterval/requestAnimationFrame/鼠标事件等
