逐帧动画介绍

1.适用于无法补间计算的动画， 相当于关键帧动画的特例 2.资源较大 3.使用steps

逐帧动画使用场景

1.动画面积小 2.动画时常短

animation 包含哪些属性

animation-name 名字 animation-duration 时间 animation-iteration-count 循环次数 animation-timing-function ：属性定义CSS动画在每一动画周期中执行的节奏。可能值为一或多个timing-function, 如：linear、steps、ease animation-play-state 播放状态 animation-fill-mode 设置CSS动画在执行之前和之后如何将样式应用于其目标。 animation-delay: 延迟时间，定义动画于何时开始，即从动画应用在元素上到动画开始的这段时间的长度。
